### 🔹 **Basic “if-else” Statement Questions**


1. **Positive or Negative Number**
   Write a python program to check whether a number is positive or negative.

answer : 

n=int(input("Enter the number : "));
if(n>0):
    print("It is a positive number ");
else:
    print("It is a negative number ");



2. **Even or Odd Number**
   Write a python program to check whether a number is even or odd.

answer : 

n=int(input("Enter the number : "));
if(n%2==0):
    print("It is a even number ");
else:
    print("It is a odd number ");


3. **Pass or Fail**
   Write a python program to check if a student has passed or failed.
   *(Pass if marks ≥ 40, otherwise Fail)*

answer : 

n=int(input("Enter the number : "));
if(n>=40):
    print("pass");
else:
    print("fail");
