1. **Check the largest of three numbers**
   Write a python program to find the largest among three numbers using nested if statements.

answer :

n=int(input("Enter the zero : "));
if(n==0):
    a=int(input("Enter the first number : "));
    b=int(input("Enter the second number : "));
    c=int(input("Enter the third number : "));
    if(a>b and a>c):
        print("Largest number : ",a);
    elif(b>a and b>c):
        print("Largest number : ",b);
    elif(c>a and c>b):
        print("Largest number : ",c);

   Use nested if to determine whether a number is positive, negative, or zero.

answer :

    a=int(input("Enter the number : "));

4. **Check whether a person is eligible to vote and also check if they are senior citizens**
   (age ≥ 18 → eligible, then check if age ≥ 60 → senior citizen)

answer :

n=int(input("Enter the age : "));
if(n>=18):
    print("Eligible for vote");
    if(n>=60):
        print("Senior citizen");
    else:
        print("Not a senior citizen");
else:
    print("Not eligible for vote");
