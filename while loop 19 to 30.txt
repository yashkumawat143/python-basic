### 🔢 **Question 18: Find Greatest Digit**

Write a python program using a while loop to find the greatest (largest) digit in a given number.

answer : 

n=int(input("Enter the number : "));
max=0;
while(n>0):
    digit=n%10;
    if(digit>max):
        max=digit;
    n=n//10;
print("maximum number = ",max);

### 🔄 **Question 22: Check Prime Number**

Write a python program using a while loop to check whether a given number is a prime number.

answer : 

n=int(input("Enter the number : "));
i=1;
count=0;
while(i<=n):
    if(n%i==0):
        count=count+1;
    i=i+1;
if(count==2):
    print("It is a prime number");
else:
    print("It is not a prime number ");

### 🧾 **Question 30: Find Sum of Even and Odd Numbers Separately (1–N)**

Write a python program using a while loop to find the sum of even and odd numbers separately from 1 to N.

answer :

i=1;
sum1=0;
sum2=0;
while(i<=n):
    if(i%2==0):
        sum1=sum1+i;
    elif(i%2!=0):
        sum2=sum2+i;
    i=i+1;    
print("The sum of even number : ",sum1);
